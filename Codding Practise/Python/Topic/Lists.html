Python List Quiz

Question 1:
What is the syntax to create an empty list in Python?
Answer: []
Hint: It uses square brackets.
Question 2:
How can you access the third element of a list named "my_list"?
Answer: my_list[2]
Hint: List indexing starts from 0.
Question 3:
What is the purpose of the len() function when applied to a list?
Answer: The len() function returns the number of elements in a list
Hint: List indexing starts from 0.
Question 4:
What is the index of the element 'apple' in the list ['banana', 'apple', 'orange']?
Answer: The index of 'apple' in the list ['banana', 'apple', 'orange'] is 1.
Hint: Use the index() method.
Question 5:
How do you add the element 'grape' to the end of the list ['apple', 'banana', 'orange']?
Answer: You can add 'grape' to the end of the list ['apple', 'banana', 'orange'] using the append() method:
my_list = ['apple', 'banana', 'orange']
my_list.append('grape')
Hint: Use the append() method.
Question 6:
What is the length of the list ['cat', 'dog', 'rabbit', 'hamster']?
Answer: The length of the list ['cat', 'dog', 'rabbit', 'hamster'] is 4.
Hint: Use the len() function.
Question 7:
How do you remove the element 'banana' from the list ['apple', 'banana', 'orange']?
Answer: You can remove 'banana' from the list ['apple', 'banana', 'orange'] using the remove() method:
my_list = ['apple', 'banana', 'orange']
my_list.remove('banana')
Hint: Use the remove() function.
Question 8:
What are the first three elements of the list ['apple', 'banana', 'orange', 'grape', 'kiwi']?
Answer: The first three elements of the list ['apple', 'banana', 'orange', 'grape', 'kiwi'] are 'apple', 'banana', and 'orange'. You can obtain them using list slicing:
my_list = ['apple', 'banana', 'orange', 'grape', 'kiwi'] ,
first_three = my_list[:3]
Hint: Use the remove() function.

How can you remove all elements from a list in Python?
